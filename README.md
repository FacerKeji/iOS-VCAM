# iOS-VCAM
苹果手机虚拟摄像头 iPhone virtual camera  
可以替换 iOS 系统摄像头画面
它处于开发中，现阶段演示地址:  [Youtube Demo](https://youtu.be/tJGXZreAW8M)

## 生效软件
使用时可随时通过快捷键 暂停/切换 替换
- 系统相机 (System Camera)
- 相机 (Message)
- 网页 (Webkit)
- QQ
- TIM
- Wechat
- TokTik
- Telegram (已知问题: 先使用拍照功能，在点右边的录像时，录像会卡顿)
- Google Translate
- Youtube
- Facebook
- Discord
- ......

## 支持系统版本

- 开发测试版本为 iOS13.3、因为手上没有其他版本所以无法真机测试
- 理论支持 iOS11.0 以上，但尚未测试
